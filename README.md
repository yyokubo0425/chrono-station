# Chrono Station

## 概要
Chrono Stationは、従業員の勤怠情報を一元管理するためのWebベースの勤怠管理システムです。  
打刻・日次・月次勤怠管理を中心に、業務ルールを意識した設計と拡張性の高いアーキテクチャを重視して開発しています。

## 主な機能
### 一般ユーザー
- 出勤・退勤打刻
- 休憩時間の管理
- 日次勤怠の確認
- 月次勤怠一覧の確認
- 打刻修正申請
- 打刻漏れ申請

### 管理者
- 勤怠データの確認
- 打刻修正申請・打刻漏れ申請の承認／却下

## 設計方針
- Controller / UseCase / Domain を分離したレイヤード設計
- 業務ルールをDomain層に集約し、UIやインフラ層から独立
- ユースケース単位で処理を分離し、機能追加に対応しやすい構成
- DTOを介したデータ受け渡しによる疎結合な設計

## 開発状況・今後の展望
日次・月次勤怠管理、打刻機能、打刻修正申請・打刻漏れ申請、および承認フローまで実装済みです。
今後は管理者向け画面の拡充に加え、勤怠データの異常を検知し、AIが自然な言語で通知する機能の実装を予定しています。
